# Public
Para realizar esta tarea lo primero después de crear una cuenta en Github y seguir los pasos solicitados.

En la terminal del ordenador he clonado el repositorio de internet, para así poder acceder a él, posteriormente he realizado un touch para poder crear el archivo de python y mostrarlo luego con el ls-l, luego lo añadido a git, con un commit en el que indicaba que agrega ese archivo, con el cual luego me pedía identificarme, lo cual he hecho con un token, en lugar de hacerlo con un nombre y usuario, ya que me parece más seguro.





He utilizado el comando:
git remote set-url origin https://Usuario:token@github.com/JuannGlezz/P-blico.git



Y finalmente he realizado un push para escribir los cambios y que se actualicen en github.
